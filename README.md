# Cloud CI/CD Demo

Week 8 OJT deliverable — a minimal static site used to build and test a GitHub Actions CI/CD pipeline (build → test → deploy to GitHub Pages).

## Structure
- `index.html` — static page deployed via the pipeline
- `package.json` — Node.js project file used for the build/test stage (Day 3)
- `.github/workflows/` — pipeline definitions (added Day 2 onward)
